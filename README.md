# Pager (2026)

Push Notifications Pager app created in Next.js 16.

Live demo [here](https://pager.mpecak.pl).

Copyright 2026 IT Pulse Mikołaj Pęcak. All rights reserved.

## Getting started

1. Create `.env.local` file and fill the data correctly
2. `npm ci` to install dependencies
3. `npm run dev` to start the app locally (http://localhost:3000 by default)
